Задание по SFML разделены на 3 папки:

Engine - движок-обертка над SFML, который используется во всех заданиях
Tasks - набор заданий, разделённый на папки (ElectricScene, etc)
Main - папка с main-файлом. Чтобы запустить нужное задание, в строчку
Scene* start = new <...>;
Где <...> - название папки с заданием

Возможно, в будущем, добавлю лаунчер всех заданий
